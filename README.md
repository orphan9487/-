# Expense Tracking Tool

這是一個用 streaklit 實作, 簡單的 "開銷追蹤工具"
允許使用者在網頁左側輸入開銷，並且在右側實時顯現花費紀錄的占比

## Installation

1. Install the required dependencies:
   ```bash
   pip install streamlit pandas plotly
   ```

## How to Run

```bash
streamlit run app.py
```
